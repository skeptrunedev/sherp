# Marp on Astro

**Markdown Presentations with Astro + MDX**

A complete replication of Marp functionality using Astro's Content Collections API

---

## Features

- Single file presentations (MDX)
- Slide splitting with horizontal rules
- Marp-style directives
- Theme support
- Navigation controls
- Keyboard shortcuts

---

## Getting Started

Create a new MDX file in `src/content/presentations/`:

```mdx
---
title: "My Presentation"
theme: "default"
paginate: true
---

# Slide 1

---

# Slide 2
```

---

## Directives

You can control slide appearance using:

- **Global directives** - In frontmatter
- **Local directives** - HTML comments per slide
- **Directive inheritance** - Slides inherit from previous
- **Scoped directives** - Use underscore prefix

## Navigation Controls

Try these keyboard shortcuts:

- Arrow keys or Space - Next/Previous slide
- Home - First slide
- End - Last slide
- O - Overview mode

---

## Overview Mode

Press `O` to see all slides at once!

Click any slide to jump to it.

Perfect for:
- Getting an overview
- Quick navigation
- Finding specific slides

## Appendix: Technical Details

This presentation system uses:

1. **Astro Content Collections** for type-safe content
2. **MDX** for rich markdown with components
3. **Custom slide parser** to split on `---`
4. **Directive parser** for Marp-style configuration
5. **CSS theming** for multiple visual styles

All in a single `.mdx` file!
